# Task Management Application

A responsive Task Management Web Application built using HTML, CSS, and JavaScript.

## 📌 Project Overview

This application helps users create, manage, track, and organize daily tasks efficiently.

Users can:
- Add tasks
- Mark tasks as completed
- Delete tasks
- Track task statistics

---

# 🚀 Features

✅ Add New Tasks  
✅ Delete Tasks  
✅ Mark Tasks as Completed  
✅ Responsive Design  
✅ Real-Time Task Statistics  
✅ Modern UI Design  
✅ Mobile Friendly
